# Archlinux源
## 默认源文件`/etc/pacman.d/mirrorlist`

## 官方源文件
1. 从官网下载源文件
```shell
wget -O /etc/pacman.d/mirrorlist https://www.archlinux.org/mirrorlist/all/
```
2. 在官方源文件中进行速度排序
```shell
cd /etc/pacman.d
cp mirrorlist mirrorlist.backup
rankmirrors -n 6 mirrorlist.backup > mirrorlist
```
